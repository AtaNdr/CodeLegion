What it does

An autonomous engineering team
in your resource group

CodeLegion turns the routine, well-scoped half of your backlog — bug fixes, dependency bumps, test additions, mechanical refactors — into reviewed pull requests. Without supervision. While your senior engineers do the work that actually requires them.

Label-driven dispatch

Drop agent-ready on a GitHub issue. Within three minutes a fresh agent claims it, posts a plan, and starts writing.

Model routing per issue

Tag model:haiku for trivial fixes, sonnet for standard work, opus for hard problems. Right tool for the job.

Wake on demand

Agent VMs spin only when work exists. They self-deallocate after 10 idle minutes. You pay for what you use — pennies per task.

Distinct agent identities

Each agent picks its own name and emoji on first boot. Visible in the dashboard. Stamped on every PR and comment. Easier reviewing.

Cost transparency, per task

Every completed task posts a cost summary on the issue: $0.07 — sonnet · 18K tokens · 4m 12s. Dashboard tracks daily and monthly totals.

Triage proposals for vague work

Under-specified issues get a structured proposal comment and wait for human approval. The agent never silently guesses at intent.

One-click Stop / Start fleet

Halt every running agent and deallocate. Persists across controller restarts. Resume restarts the loop. Hard cost ceiling, one click.

Self-hosted. No SaaS lock-in

One Azure Web App in your subscription. GitHub App with your installation token. Secrets never leave your tenant.

Live dashboard

Per-VM state, reconcile history, cost breakdown, full audit trail. Auto-refreshes only the fleet section — the rest stays stable as you read.

Get running

Three steps. ~15 minutes.

CodeLegion provisions its own infrastructure. Give it three things and walk away.

1

Anthropic API key

Generate at console.anthropic.com. Configure billing. You'll paste it into the setup wizard.

~2 minutes
2

Azure Web App + outbound

Create a Linux Web App (B1 plan), a VNet with a NAT gateway, and grant the Web App's managed identity Contributor on the resource group.

~10 minutes
3

Deploy this repo

Point Azure Deployment Center at github.com/AtaNdr/CodeLegion. Walk the wizard. CodeLegion creates the rest — subnet, NSG, agent VMs.

~3 minutes

Why CodeLegion

Built for the way
engineering teams already work

No new tools to learn. No SaaS. No data leaving your tenant. Every action visible on GitHub and reviewable by a human before it merges.

Reviewed before merge. Every change opens a PR. Branch protection enforces 1 review + CODEOWNERS. Nothing ships unsupervised.
Tests per acceptance criterion. Every PR body maps each acceptance criterion to its covering test. No "trust me" merges.
Pay per task, not per seat. Anthropic tokens + Azure infra. Typical scoped issue resolves for under 30 cents. Hard caps everywhere.
Secrets stay in your tenant. GitHub App PEM, Anthropic key, webhook secret — all in your Web App's App Settings. No third-party broker.
Audit trail is GitHub itself. Decisions, plans, test results, cost — every step lives in issue comments and PR diffs. No separate audit log to maintain.
MIT licensed. Yours to fork. Open source, single-instance by design, small enough to fully read in an afternoon (~5K LOC controller).
